# CycleHop

A Swift Playground app for finding London bike-sharing stations. Search for a destination and Cycle Hop recommends the best nearby docking stations based on distance, bike availability, and direction of travel.

## Features

- Interactive map with live station markers showing availability
- Smart recommendation engine that scores stations by proximity, availability, and route alignment
- Search powered by MapKit for any London location
- Adaptive layout: bottom sheet on iPhone, sidebar on iPad/Mac

## Requirements

- iOS 18.1+
- Swift Playgrounds or Xcode

## Getting Started

Open `My App.swiftpm` in Swift Playgrounds or Xcode and run on a simulator or device.
